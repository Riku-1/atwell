## golang-api-sample

### Description
This is an example of api by golang.

### Features
- Clean architecture
- OpenAPI(Swagger)
- CI
- Testing
- Docker

### API Documentation
API documentation is in docs folder.  
It is automatically generated by swag and created by following commands.
```shell
swag init --parseDependency --parseInternal
```
